# Count Button

Aplicação simples de contador desenvolvida em Python.

## Requisitos

- Python 3.x

## Instalação

1. Clone o repositório
2. Crie e ative o ambiente virtual:
```bash
python -m venv .venv
.venv\Scripts\activate
```

3. Instale as dependências:
```bash
pip install -r requirements.txt
```

## Execução

```bash
.venv\Scripts\python.exe counter_app.py
```

## Testes

```bash
.venv\Scripts\python.exe -m pytest tests/
```
